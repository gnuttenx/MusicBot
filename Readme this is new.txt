Developed from the original:review version by Just-Some-Bots. 
Codes taken from different users and put together.

Now playing [np] added a little visual bar for progress.

New commands:
playnow - same as play, but replace the current song and starts playing directly.
promote # - Place song to first spot in queue. # = The songs place in the playlist.
next - The same as skip.
remove # - Remove song from playlist. # = The songs place in the playlist.
stop - The same as pause
